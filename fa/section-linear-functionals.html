Section 4.1 Linear functionals: definition, examples, properties

A linear functional on a vector space \(V\) is a function \(L\colon V\to \C\) such that \(L(\alpha x+\beta y) = \alpha L(x) + \beta L(y)\) for all \(x, y\in V\) and all \(\alpha, \beta \in \C\text{.}\) Examples:

  1. Coordinate functionals on \(\ell^2\text{,}\) that is \(L_k(x) = x(k)\)
  2. On \(c_{00}\) one can define the sum \(\Sigma(x) = \sum_k x(k)\text{,}\) but this does not work on \(\ell^2\)
  3. On \(L^2([a,b])\text{,}\) the integral \(I(f) = \int_a^b f(x)\,dx\) is a linear functional.

The sum functional \(\Sigma\) is not continuous on \(c_{00}\text{.}\) Indeed, let \(x_n(k)=1/n\) when \(k\le n \) and zero otherwise. Then \(\sigma(x_n)=1\) for every \(n\text{,}\) but \(x_n\to 0\) in \(c_{00}\text{,}\) and of course \(\sigma(0)=0\text{.}\)

A Hamel basis is a maximal linearly independent set. Every vector space has one, although in some infinite dimensional spaces, including \(\ell^2\text{,}\) there is no explicit form of such a basis; we infer its existence from Zorn's Lemma. Using a Hamel basis in \(\ell^2\text{,}\) one can extend the sum functional \(\Sigma\) to \(\sigma\colon \ell^2\to \C\) (not in any explicit way), thus obtaining a discontinuous linear functional on a Hilbert space. There are no explicit examples of this kind.

The following properties of a linear functional \(L\) on a Hilbert space \(\Hi\) are equivalent:

  1. \(L\) is continuous on \(\Hi\)
  2. \(L\) is continuous at \(0\)
  3. \(L\) is bounded, meaning there exists a constant \(C\) such that \(|L(x)|\le C\|x\|\) for all \(x\in \Hi\)

The norm of a bounded linear functional is the smallest \(C\) that works in the definition of boundedness: equivalently,

\begin{equation*} \|L\| = \sup_{x\ne 0} \frac{|L(x)|}{\|x\|} = \sup_{\|x\|=1} |L(x)| \end{equation*}

What are the norms of coordinate functionals on \(\ell^2\) and the integral functional on \(L^2([a, b])\text{?}\)

The kernel of a linear functional is \(\ker L = L^{-1}(0) = \{x\colon L(x)=0\}\text{.}\) This is always a linear manifold. It is a linear subspace if and only if \(L\) is continuous.
