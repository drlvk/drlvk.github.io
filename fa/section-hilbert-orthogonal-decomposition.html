Section 2.2 Orthogonal decomposition

Two vectors are orthogonal if their inner product is zero. The Pythagorean theorem applies: \(\|x+y\|^2=\|x\|^2+\|y\|^2\) when \(x, y \) are orthogonal.

For a subset \(A\text{,}\) its orthogonal complement \(A^\perp\) is the set of vectors orthogonal to every element of \(A\text{.}\) Note that this is a linear subspace even if \(A\) is not.

Suppose \(\Mi\) is a linear subspace of \(\Hi\text{.}\) We'd like to decompose every element \(x\in\Hi\) as \(x'+x''\) with \(x'\in \Mi\text{,}\) \(x''\in \Mi^\perp\text{.}\)

Geometric idea: let \(x'\) be the nearest element of \(\Mi\) to \(x\text{.}\) If such a thing exists, then we are guaranteed that \(x-x'\in \Mi^\perp\) (take \(u\in \Mi\) such that \(\inn{u,x-x''}>0\text{,}\) and consider \(x'+tu\)).

Why does such \(x'\) exist? You may recall about distance to compact set being attained. But we don't have compactness here. Completeness will have to do, but we need a Cauchy sequence.

So, let \(d\) be the distance from \(x\) to \(\Mi\) and pick \(x_n\) such that \(\|x-x_n\|< d+1/n\text{.}\) Can \(x_n-x_m\) be large? This is a 2D picture, really: both \(x_n,x_m\) lie within a disk of radius \(d+1/n\text{,}\) and the chord stays at distance \(d\) from the center. So, the chord must be short. To justify this, use the parallelogram identity \(\|a+b\|^2 + \|a-b\|^2 = 2\|a\|^2 + 2\|b\|^2\text{.}\)

So, the sequence is Cauchy, hence convergent. We get \(x'\) and with it \(x''\text{,}\) completing the proof of existence of orthogonal decomposition \(x=x'+x''\text{.}\)

Note that this decomposition is unique, because if we also had \(x=y'+y''\) then \(x'+x'' = y'+y''\) implies \(x'-y' = x''-y'' \in \Mi\cap \Mi\perp\text{.}\)

We can define orthogonal projection onto \(\Mi\) by \(P_{\Mi} x = x'\text{.}\) Some claims to consider:

  • \(\displaystyle \|P_{\Mi}(x)-P_{\Mi}(y)\|\le \|x-y\|\)
  • \(P_{\Mi}\) is linear (formal definition will be given later)

Bonus problem: for every fixed \(y\text{,}\) the function \(f(x) = \inn{x,y}\) is continuous on \(\Hi\text{.}\)
