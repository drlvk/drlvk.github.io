Section 5.1 Orthonormal sets and Bessel's inequality

An orthonormal set in an inner product space is a set \(\Ei\) such that any two distinct elements of \(\Ei\) are orthogonal to each other and each element has unit norm. By the Pythagorean theorem, \(\|e_1-e_2\|=\sqrt{2}\) for any two distinct elements \(e_1, e_2\in \Ei\text{.}\)

A maximal orthonormal set is called an orthonormal basis. (Exists by Zorn's lemma).

Every orthonormal set is linearly independent. Hence, in a finite-dimensional space such as \(\C^n\text{,}\) every orthonormal set is finite. The standard basis vectors form a countable orthonormal set in \(\ell^2\) (indeed, this is a orthonormal basis). In a separable Hilbert space (one with a countable dense subset), any orthonormal set is countable. Both \(\ell^2\) and \(L^2([a, b])\) are separable. Non-separable Hilbert spaces exist but they do not arise often, so for now we focus on countable orthonormal sets.

An orthonormal basis for \(L^2([0,1])\) takes more work. One such basis is the Fourier basis \(\{\exp(2\pi i n t)\colon n\in \mathbb Z\}\text{.}\) Orthonormality is easy, but the maximality is nontrivial. Use the Stone-Weierstrass theorem: if an algebra is closed under conjugation, separates elements and contains constant function, then it is dense in \(C[0,1]\text{.}\) Hence, the set of exponential polynomials is dense in \(L^2([0,1])\text{.}\)

Suppose that \(\{e_k\colon n\in \mathbb N\}\) is an orthonormal set in a Hilbert space \(\Hi\text{.}\) Bessel's inequality states that

\begin{equation} \sum_{k=1}^\infty \abs{\inn{h,e_k}}^2 \le \|h\|^2,\quad \forall h\in\Hi\label{eq14-bessel}\tag{5.1.1} \end{equation}

Proof: consider the difference between \(h\) and partial sum

\begin{equation} h_n = \sum_{k=1}^n \inn{h,e_k} e_k\label{eq14-projection}\tag{5.1.2} \end{equation}

Since \(h-h_n\) is orthogonal to \(e_1,\dots, e_n\text{,}\) it is orthogonal to \(h_n\text{.}\) By the Pythagorean theorem

\begin{equation} \|h\|^2=\|h-h_n\|^2+\|h_n\|^2\label{eq14-pythagorean}\tag{5.1.3} \end{equation}

hence

\begin{equation} \|h_n\|^2 = \sum_{k=1}^n |\inn{h, e_k}|^2 \le \|h\|^2\label{eq14-norm}\tag{5.1.4} \end{equation}

The conclusion follows.

For the future, note that \(\|h-h_n\|\) is the distance from \(h\) to the span of \(\{e_1, \dots, e_n\}\) because \(h_n\) is its projection onto this span.
