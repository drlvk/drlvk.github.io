More generally, given a measure space \(\Omega\) with a measure \(\mu\text{,}\) we can define the space \(L^2(\Omega)\) as the set of equivalence classes of Lebesgue measurable functions \(f\colon \Omega\to\mathbb C\) such that \(|f|^2\) is integrable (\(\int_\Omega |f|^2\,d\mu < \infty\) is finite). The unit interval with the Lebesgue measure is a standard example: \(L^2([0, 1])\text{.}\)
