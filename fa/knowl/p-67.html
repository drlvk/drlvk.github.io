More generally, unconditional convergence can be defined for a series \(\sum_{i\in I} x_i\) where \(x_i\in \Hi\) and \(I\) is an abstract set of indices (without any order). Its unconditional sum is \(S\) if for every \(\epsilon\) there is a finite set \(A\subset I\) such that \(\left\|\sum_{i\in B}x_i - S\right\|<\epsilon\) for every finite set \(B\) containing \(A\text{.}\)
