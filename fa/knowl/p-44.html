A linear manifold \(A\) is dense in \(\Hi\) if and only if \(A^\perp = \{0\}\text{.}\) Indeed, if \(A^\perp\) contains a nonzero vector \(x\text{,}\) then we have \(\|x-a\|\ge \|x\|\) for all \(a\in A\text{,}\) so \(A\) is not dense. Conversely, if \(A\) is not dense, then \(\overline{A}\) is a proper linear subspace, which has a nontrivial orthogonal complement.
