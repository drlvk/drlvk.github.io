A linear functional on a vector space \(V\) is a function \(L\colon V\to \C\) such that \(L(\alpha x+\beta y) = \alpha L(x) + \beta L(y)\) for all \(x, y\in V\) and all \(\alpha, \beta \in \C\text{.}\) Examples:

  1. Coordinate functionals on \(\ell^2\text{,}\) that is \(L_k(x) = x(k)\)
  2. On \(c_{00}\) one can define the sum \(\Sigma(x) = \sum_k x(k)\text{,}\) but this does not work on \(\ell^2\)
  3. On \(L^2([a,b])\text{,}\) the integral \(I(f) = \int_a^b f(x)\,dx\) is a linear functional.
