An orthonormal set in an inner product space is a set \(\Ei\) such that any two distinct elements of \(\Ei\) are orthogonal to each other and each element has unit norm. By the Pythagorean theorem, \(\|e_1-e_2\|=\sqrt{2}\) for any two distinct elements \(e_1, e_2\in \Ei\text{.}\)
