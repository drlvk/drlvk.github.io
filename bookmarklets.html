Bookmarklets

To use, drag the link in bold to your bookmark bar.

Open the page with WebWork problem about which you got an email. WebWork email: When executed on a Gmail page or Outlook Web Access page with a student's email that contains a link to a WeBWork problem, it opens WebWork page with student's work. It is assumed that the student emails from their syr.edu account. The script locates the link to WebWork problem and the student's email address, and creates a URL of the form "...course/set/problem/?showCorrectAnswers=1&effectiveUser=username", which it then opens in a new tab. source
