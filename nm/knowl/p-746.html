There is a discrete version of the Fourier series theory outlined in SectionÂ 25.2. In it we fix an integer \(n\) and replace a function \(f\colon [0, 1]\to \mathbb C\) by its sampled values \(z_k = f(k/n)\text{,}\) \(k=0, 1, \dots, n-1\text{.}\) The discrete version of inner product (25.2.1) is

\begin{equation} \langle f, g \rangle_d = \sum_{\ell=0}^{n-1}f(\ell/n)\overline{g(\ell/n)} = \sum_{\ell=0}^{n-1} z_\ell \overline{w_\ell}\tag{25.3.1} \end{equation}

The functions \(f_k = \exp(2\pi i k)\text{,}\) \(k=0, \dots, n-1\text{,}\) are orthogonal with respect to this inner product. (Not orthonormal, as \(\langle f_k, f_k\rangle = n\text{.}\)) Note we only need \(n\) of these functions (why?).
