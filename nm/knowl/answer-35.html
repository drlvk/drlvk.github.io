Answer 5.3.2.1. 
function y = p(x, a, n)
    y = ones(size(x));
    for k = 1:n
        y = y + x.^k/(a+k-1);
    end
end

The command size(x) returns the size of \(x\) as the number of rows and columns. Consequently, ones(size(x)) means creating a vector or matrix of the same size as x, but with all entries 1. If we started with y = 1 instead, the computation would still work because adding 1 to a vector like x.^k means adding 1 to each component of the vector. However, when n = 0 the loop does not run at all and the output would be a scalar 1 instead of a vector, which is a bug.

One could also start with y = zeros(size(x)) and run the loop for k=0:n; the result is the same.
