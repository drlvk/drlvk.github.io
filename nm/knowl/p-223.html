There is another logical choice of solution in a problem like this: one could pick the point on the line that is closest to the origin \((0, 0, 0)\text{,}\) that is the solution of smallest norm. This is the approach that Octave uses: when given ExampleÂ 6.3.1 it would produce [-4.5556; 0.1111; 4.7778]. One can get this solution with Matlab by using pinv(A)*b, which results in [-4.5556; 0.1111; 4.7778].
