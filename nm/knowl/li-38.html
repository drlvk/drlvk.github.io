Item 2.7.1.a.
Find the first 30 Fibonacci numbers using Matlab's linear solver A\b. To achieve this, you will first need to create a \(30\times 30\) matrix of the linear system and then create a column vector b where the first two coordinates are \(1\) and the rest are \(0\text{.}\) The commands eye, diag, ones, and zeros will help.
