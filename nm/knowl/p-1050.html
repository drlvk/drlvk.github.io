To help with visualization, consider the case \(m=2\) when the explanatory variables \((x_1, x_2)\) are more conveniently labeled \((x, y)\text{.}\) We are looking for a line \(y = ax+b\) such that the points of one category are above it, and the points of other category are below it. If such a line exists, it is not unique, so we want to choose it to maximize the width of separation. That is, we look for maximal \(w\ge 0\) such that \(y_j\ge ax_j + b + w\) in the first set (above the line) and \(y_j\le ax_j +b - w\) in the second set (below the line). This is a linear programming problem, with three variables \(a, b, w\text{,}\) objective function \(w\text{,}\) and the linear constraints stated above.
