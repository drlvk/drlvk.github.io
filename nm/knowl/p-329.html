Formally, the linear approximation to \(f\) at a point \(x\) is \(f(x+h) \approx f(x) + f'(x)h\text{.}\) Equating the right hand side to 0 we find \(h = -f(x)/f'(x)\) which means that the point where the tangent line at \(x\) meets the line \(y=0\) is \(x + h = x - f(x)/f'(x)\text{.}\) Therefore, we can express Newton's method as iteration of the function

\begin{equation*} g(x) = x - \frac{f(x)}{f'(x)} = \frac{xf'(x) -f(x)}{f'(x)} \end{equation*}

Note that any point where \(f(x) = 0\) and \(f'(x)\ne 0\) is a fixed point of \(g\text{.}\) In fact, it is a super-attracting point of \(g\) because

\begin{equation*} g'(x) = 1 - \frac{f'(x)f'(x) - f(x)f''(x)}{(f'(x))^2} = \frac{f(x)f''(x)}{(f'(x))^2} \end{equation*}

which is zero when \(f(x) = 0\text{.}\)
