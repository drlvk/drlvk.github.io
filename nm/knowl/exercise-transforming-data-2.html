Exercise 2.

The number of active Covid cases at SU during October 1 - October 23 is given below.

y = [5 5 4 5 9 25 45 65 77 87 87 89 97 101 90 74 57 26 20 13 12 12 13]';

The x-values can be x = (1:23)'. Try to fit a “bell-shaped” Gaussian function \(y = \exp(Ax^2+Bx+C)\) (where \(A\lt 0\)) to the data. This can be done by the logarithmic transform yt = log(y) followed by fitting a quadratic function to transformed data in the usual way, via the matrix X = x.^(0:2). Having found the parameters beta, we get a suitable function f = @(x) exp(x.^(0:2)*beta) which should be plotted together with the data points.
