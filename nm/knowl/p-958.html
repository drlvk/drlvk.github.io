Sometimes the nature of the problem requires the variables \(x_k\) to be nonnegative, which leads to

\begin{equation} \mathbf c^T \mathbf x\to \min \quad \text{where } A\mathbf x \le \mathbf b, \quad \mathbf x \ge 0\tag{35.1.2} \end{equation}

Or, one may have linear equations instead of inequalities:

\begin{equation} \mathbf c^T \mathbf x\to \min \quad \text{where } A\mathbf x = \mathbf b, \quad \mathbf x \ge 0\tag{35.1.3} \end{equation}
