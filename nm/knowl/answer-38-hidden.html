x = [-1, 0, 2, 3, 4, 7]'; 
y = [9, 8, 5, 5, 3, 1]';
A = [x.^2, x, x.^0];
z = A\y;
xx = linspace(min(x), max(x), 500)';
yy = [xx.^2, xx, xx.^0]*z;
hold on
plot(xx, yy)
plot(x, y, 'r*')
hold off

Explanation: the only modification that we need is to add a column with the squares of x-values. In general, this process allows us to fit a function of the form \(c_1 f_1(x) + c_2 f_2(x) + \cdots + c_n f_n(x) \) where \(f_1, f_2, \dots, f_n\) are given functions and \(c_1, c_2, \dots, c_n\) are coefficients (parameters) to be determined.

Note that the parabola we get is not very different from a line. The line was already a good fit, and we did not gain much from using a more complicated curve. Later we will consider the issue of model selection: how to decide which model is more appropriate (for example, linear or quadratic).
