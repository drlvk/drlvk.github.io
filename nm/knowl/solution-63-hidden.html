We maximize \(w\) subject to constraints \(y_j\ge ax_j^2 + bx_j + c + w\) in the first group and \(y_j\le ax_j^2 + bx_j + c - w\) in the second. Rewrite the constraints as

\begin{equation*} ax_j^2 + bx_j + c + w \le y_j \end{equation*}

in the first group and

\begin{equation*} -ax_j^2 - bx_j - c + w \le -y_j \end{equation*}

in the second. So, given the vectors xt, yt describing the first group and xb, yb for the second, we can form a system of linear inequalities \(A\mathbf x\le \mathbf b\text{.}\)

xt = [2; 4; 7];
yt = [3; 6; 4];
xb = [3; 5];
yb = [4; 5];
A = [xt.^2 xt xt.^0 xt.^0; -xb.^2 -xb -xb.^0 xb.^0];
b = [yt; -yb];
opt = linprog([0; 0; 0; -1], A, b);

t = linspace(min([xt; xb]), max([xt; xb]), 1000);
y = opt(1)*t.^2 + opt(2)*t + opt(3);
plot(xt, yt, 'b*', xb, yb, 'r*', t, y, 'k')
