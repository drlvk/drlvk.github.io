Linear algebra says that an \(n\times n\) matrix \(A\) is singular (non-invertible) if and only if \(\det A = 0\text{,}\) if and only if \(\rank A \lt n\text{.}\) But in computational practice these two tests for invertibility can give different results. Consider the diagonal matrix with entries \(10^{16}\) and \(1\text{.}\)

A = [1e16 0; 0 1];
disp(det(A))
disp(rank(A))

Matlab says the determinant is \(10^{16}\) which makes sense: it is the product of diagonal entries. Yet, it also says the rank is \(1\text{.}\) Why?
