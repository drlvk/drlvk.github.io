(Theoretical) Show that no fixed value \(\beta > 0\) can make the gradient descent converge for the function \(f(x) = |x|^{3/2}\text{.}\)

(Note that \(f\) is differentiable, with \(f'(x) = (3/2)\sqrt{x}\) for \(x\ge 0\) and \(f'(x) = -(3/2)\sqrt{-x}\) for \(x\le 0\text{.}\))
