function z = agm(x, y, tol) 
   while abs(x - y) >= tol
       a = (x + y)/2;
       g = sqrt(x*y);
       x = a;
       y = g;
   end
   z = x;
end

Note that the result has to be assigned to whichever function variable was designated as output on th first line of the function: in this case, z.
