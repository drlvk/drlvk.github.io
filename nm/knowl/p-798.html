y = [5 5 4 5 9 25 45 65 77 87 87 89 97 101 90 74 57 26 20 13 12 12 13]';

The x-values can be x = (1:23)'. Considering the shape of this data, one may want to model it by a rational function of the form \(y = 1/(Ax^2 + Bx+ C)\) (to see why, think of how \(y=1/(x^2+1)\) looks.) Following the logic of Example 28.3.1, rewrite the equation as \(Ax^2 y + Bxy + Cy = 1\) and find the optimal \(A, B, C\) using least squares. Unlike in Example 28.3.1, we do not need an implicit plot here since the function \(y = 1/(Ax^2 + Bx+ C)\) can be plotted directly. Plot it together with the data points. How well does it fit?
