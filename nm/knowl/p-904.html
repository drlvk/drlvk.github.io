However, it would be pointless to replace \(C\) by \(R\) if this did not make the function smaller. So the replacement only happens if \(f(R)\lt f(C)\text{.}\) Otherwise the program stops and reports the midpoint \((A+B+C)/3\) of the last computed triangle as an approximate point of minimum. This is a “reflection-only” version of the Nelder-Mead method.
