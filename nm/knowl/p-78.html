Sometimes we can apply an elementwise array operation to two arrays of different sizes. The rule is that the array sizes must match, but number 1 matches any number. For example, the following commands work, and produce a 3Ã—5 matrix of 2s:

  • ones(3, 1) + ones(3, 5)
  • ones(1, 5) + ones(3, 5)
  • ones(1, 5) + ones(3, 1)
