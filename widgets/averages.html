Averages

An average is a sum shared equally — the balancing point of a set of numbers. The same idea locates the center of an interval and, run backward, recovers a total from an average.

average = sum ÷ count

With two numbers the average lands exactly halfway between them — which is why the center of an interval (a, b) is (a + b) ÷ 2, and its half-width is (b − a) ÷ 2, so the interval is center ± half-width. Run the identity backward and a total appears: sum = average × count.

sum — all the values added up count — how many values average — the sum shared equally
